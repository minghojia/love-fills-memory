# 🧩 COMPONENT COMPOSITION｜元件組合

## 1. Greeting Section

🇺🇸
Simple emotional entry point.

No navigation.

No buttons.

---

🇹🇼
簡單情緒入口。

沒有導航、沒有按鈕。

---

## 2. Main Input Area (PRIMARY)

🇺🇸

This is the most important element on the screen.

### Features:
- Large rounded input box
- Tap = text input
- Hold = voice input
- Icon = photo input

### Behavior:
- Auto-save after pause
- No submit button
- No form validation

---

🇹🇼

這是畫面最重要的元件。

### 功能：
- 大型圓角輸入框
- 點擊＝文字
- 長按＝語音
- 圖示＝照片

### 行為：
- 停頓即自動儲存
- 無提交按鈕
- 無表單驗證

---

## 3. Recent Timeline

🇺🇸

Simple vertical list.

### Rules:
- Last 3–5 entries only
- No filters
- Tap to view detail

---

🇹🇼

簡單垂直列表。

### 規則：
- 僅顯示最近 3–5 筆
- 不做篩選
- 點擊進入詳情

---

## 4. AI Response Panel

🇺🇸

Appears after entry is saved.

### Structure:

1. Summary
2. Insight
3. Emotional Support

### Behavior:
- fade in
- auto-dismiss optional

---

🇹🇼

記錄完成後出現。

### 結構：

1. 摘要
2. 洞察
3. 情緒支持

### 行為：
- 淡入顯示
- 可自動消失
