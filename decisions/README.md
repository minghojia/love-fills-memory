# Love Fills Memory — Decision Log

This folder records all important product, design, and strategy decisions.

We do this because:

We don’t just build features.
We build understanding.

---

## Why this exists

Over time, we will forget why we made decisions.

This system helps us remember:

- what we decided
- why we decided it
- what we considered
- what we rejected

---

## Decision format

Each decision file must include:

1. Date
2. Decision summary
3. Context
4. Options considered
5. Reasoning
6. Outcome
7. Revisit conditions

---

## Core principle

We don’t just optimize for speed.

We optimize for clarity over time.
